使用說明

複製 CSS 後，如何在你的網站上使用

快速開始

1

選擇氛圍

在首頁點擊任一氛圍圓圈，預覽效果。

2

複製 CSS

點擊「複製 CSS（含動畫）」或「複製 CSS（純漸層）」按鈕。

3

貼上 CSS

將複製的內容貼到你的 CSS 檔案中。

4

加入 HTML

在你的 HTML 加入以下結構：

<!-- 放在 body 開頭 -->
<div class="aura-background">
  <div class="blob blob-1"></div>
  <div class="blob blob-2"></div>
  <div class="blob blob-3"></div>
  <div class="blob blob-4"></div>
</div>

<!-- 你的網頁內容放在這裡 -->
<main>
  ...
</main>
5

完成！

重新整理頁面，你的網站就有漂亮的漸層背景了。

兩種版本差異

含動畫版

  • 光球會緩慢漂浮移動
  • 視覺效果更生動
  • 檔案較大（含 @keyframes）
  • 適合：Landing Page、作品集

純漸層版

  • 光球固定不動
  • 效能較好
  • 檔案較小
  • 適合：Dashboard、文章頁

完整範例

HTML 檔案

<!DOCTYPE html>
<html lang="zh-Hant">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>我的網站</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <!-- Aura 背景 -->
  <div class="aura-background">
    <div class="blob blob-1"></div>
    <div class="blob blob-2"></div>
    <div class="blob blob-3"></div>
    <div class="blob blob-4"></div>
  </div>

  <!-- 你的內容 -->
  <main>
    <h1>歡迎來到我的網站</h1>
    <p>這裡是內容...</p>
  </main>
</body>
</html>

CSS 檔案

將從 Aura Moods 複製的 CSS 貼到 style.css，然後加上你自己的樣式：

/* ===== 從 Aura Moods 複製的 CSS ===== */
/* （貼在這裡） */

/* ===== 你自己的樣式 ===== */
main {
  position: relative;  /* 讓內容顯示在背景之上 */
  z-index: 1;
  padding: 2rem;
  color: white;
}

常見問題

為什麼看不到背景？

  • 確認 HTML 結構正確（4 個 blob div）
  • 確認 CSS 已正確載入
  • 確認沒有其他元素蓋住背景

背景顏色可以改嗎？

可以！修改 CSS 中的 --blob-1 到 --blob-4 變數即可：

:root {
  --blob-1: rgb(255, 100, 100);  /* 改成你要的顏色 */
  --blob-2: rgb(100, 255, 100);
  /* ... */
}

動畫太快/太慢怎麼辦？

修改 CSS 中各 blob 的 animation 時間：

.blob-1 {
  animation: float-1 18s ease-in-out infinite;
  /*                 ↑ 改這個數字，越大越慢 */
}

想減少效能消耗？

  • 使用「純漸層版」（無動畫）
  • 減少 blur 值：--blur-amount: 80px;
  • 只保留 2 個 blob（刪除 blob-3、blob-4）

Tailwind Token 說明

如果你使用 Tailwind CSS，可以複製「Tailwind Token」按鈕的內容。

注意

Tailwind Token 只包含顏色定義，不包含動畫。你仍需要手動加入 @keyframes 動畫（如需要）。

Tailwind v4

直接將 @theme 區塊加入你的 CSS：

@theme {
  --color-aura-bg: oklch(15% 0.03 270);
  --color-aura-blob-1: oklch(55% 0.18 260);
  /* ... */
}

Tailwind v3

將顏色加入 tailwind.config.js：

module.exports = {
  theme: {
    extend: {
      colors: {
        'aura-bg': 'rgb(15, 15, 35)',
        'aura-blob-1': 'rgb(80, 120, 220)',
        // ...
      }
    }
  }
}
